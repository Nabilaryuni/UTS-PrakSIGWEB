Fakultas di UGM

Pengenalan Praktikum

Praktikum SIG Web merupakan salah satu mata kuliah yang ada di Departemen Teknik Geodesi. Dalam pembelajarannya, praktikum ini mengajarkan mengenai tutorial membuat web dengan informasi geografis.

Minggu Pertama: HTML & CSS

Pada minggu pertama, pembelajaran dimulai dengan membuat halaman web sederhana menggunakan HTML dan CSS.

Minggu Kedua: Curriculum Vitae

Kemudian dilanjutkan membuat Curriculum Vitae menggunakan platform wix.com.

Minggu Ketiga: Web Interaktif Dasar

Minggu ketiga dilakukan pembuatan web kembali namun mengarah pada perhitungan dasar seperti hitung faktorial, banyak bilangan prima, dan banyak bilangan Fibonacci. Selain itu juga dibuat web untuk memvalidasi kebenaran password dan umur.

Minggu Keempat: Bootstrap & GitHub

Minggu keempat praktikum membahas penggunaan Bootstrap dalam pembuatan web. Hasil web yang sudah dikerjakan kemudian diupload pada GitHub.

Minggu Kelima: Integrasi Leaflet

Minggu kelima, pembelajaran meningkat dan mulai membahas penambahan peta pada web menggunakan Leaflet.

Minggu Keenam: Leaflet + GeoJSON

Minggu terakhir, pertemuan tetap menggunakan Leaflet namun digabungkan dengan data GeoJSON. Sehingga dapat ditampilkan secara interaktif nama-nama kecamatan pada area yang dipilih (dalam hal ini wilayah Yogyakarta).
