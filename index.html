Fakultas di UGM

Praktikum SIG Web merupakan salah satu mata kuliah yang ada di Departemen Teknik Geodesi. Dalam pembelajarannya , praktikum ini mengajarkan mengenai tutorial membuat web namun dengan informasi geografis . Pada minggu pertama, pembelajaran di mulai dengan membuat halaman web sederhana dengan menggunakan HTML dan CSS. Kemudian dilanjutkan membuat Curicullum Vitae menggunakan wix.com . Minggu ketiga, dilakukan pembuatan web kembali namun mengarah pada perhitungan dasar seperti hitung faktorial, banyak bilangan prima, dan banyak bilangan Fibbonanci .Selain itu juga dibuat web untuk memvalidasi kebenaran password dan umur Minggu keempat praktikum membahas mengenai penggunaan bootsrap dalam pembuatan web. Hasil web yang sudah dikerjakan kemudian di upload pada github. Minggu kelima ,tingkat pembelajaran meningkat dan mulai membahas mengenai penambahan peta pada web menggunakan leaflet. Minggu terakhir pertemuan tetap menggunakan leaflet, namun digabungkan dengan data Geojson yang ada. Sehingga dapat ditampilkan secara interaktif nama-nama kecamatan yang ada pada area yang dipilih (pada praktikum ini dipilih area yogyakarta).

Temukan Fakultasmu di UGM !
